Página de práctica con HTML y CSS

Ejemplos y explicaciones de etiquetas básicas y avanzadas

Etiquetas básicas

<h1> - Títulos

Este es un título grande

La etiqueta <h1> se usa para títulos principales. Existen hasta <h6>, cada uno con un tamaño menor.

<p> - Párrafo

Hola, soy un párrafo de ejemplo.

La etiqueta <p> sirve para escribir bloques de texto corrido, como párrafos de un documento.

<a> - Enlace

Ir a Google
La etiqueta <a> crea enlaces. El atributo href define la URL y target="_blank" abre el enlace en una nueva pestaña.

<img> - Imagen

La etiqueta <img> inserta imágenes. El atributo src indica la dirección y alt es el texto alternativo si la imagen no carga.

<ul> y <ol> - Listas

  • Elemento 1
  • Elemento 2
  1. Primero
  2. Segundo
<ul> crea listas sin orden (viñetas), mientras que <ol> genera listas ordenadas con números. Dentro se usan <li> para cada ítem.

<table> - Tablas

Nombre Edad
Juan 20
La etiqueta <table> crea tablas. Se usan <tr> para filas, <th> para encabezados y <td> para celdas.

Etiquetas nuevas

<video>

<video> inserta videos en la página. El atributo controls añade botones de reproducción y volumen.

<audio>

<audio> permite reproducir sonidos o música en la web. Igual que el video, controls añade controles básicos.

<canvas>

<canvas> crea un lienzo donde se puede dibujar usando JavaScript, como gráficos, formas o animaciones.

<figure> y <figcaption>

<figure> agrupa imágenes, gráficos o diagramas, y <figcaption> añade una descripción asociada.

<progress>

<progress> representa el avance de una tarea en curso, con valores de progreso (ejemplo: 70 de 100).

<meter>

70%
<meter> muestra una medida dentro de un rango, como niveles de batería o calificación de desempeño.

<details> y <summary>

Haz clic para ver más

Este texto estaba oculto.

<details> permite ocultar y mostrar información, y <summary> define el texto que se ve como título desplegable.

<mark>

Ejemplo: Esto es un texto resaltado.

La etiqueta <mark> resalta un fragmento de texto, como si lo marcaras con un resaltador.
