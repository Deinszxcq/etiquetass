Página de práctica con HTML y CSS

En este documento se muestran etiquetas básicas y también nuevas etiquetas con ejemplos y explicaciones.

Etiquetas básicas

<h1> - Títulos

Sirve para colocar encabezados. Ejemplo:

Este es un título grande

<p> - Párrafo

Se usa para texto normal. Ejemplo:

Hola, soy un párrafo de ejemplo.

<a> - Enlace

Sirve para crear vínculos. Ejemplo: Ir a Google

<img> - Imagen

<table> - Tablas

Permiten mostrar datos en filas y columnas:
